Detailed Description

MDS (行情数据服务系统) 异步接口库

Version
0.15.9.1 2019/07/01
  • 增加支持异步行情接收的API接口库
    • 支持异步行情接收
    • 支持自动的连接管理 (自动识别异常并重建连接)
  • 样例代码参见:
    • samples/01_mds_async_tcp_sample.c
0.15.9.4 2019/11/19
  • 支持在运行过程中重新指定行情订阅条件
  • 调整 OnConnect 回调函数的返回值约定
See also
F_MDSAPI_ASYNC_ON_DISCONNECT_T
Version
0.15.10 2019/12/20
  • 重构异步API, 统一交易和行情的异步接口
0.15.11 2020/05/29
  • 增加辅助的异步API接口
    • MdsAsyncApi_IsAllTerminated, 返回异步API相关的所有线程是否都已经安全退出
  • 删除并不适用于行情异步API的密码修改接口
    • MdsAsyncApi_SendChangePasswordReq
0.15.11.6 2020/07/16
  • 增加辅助的异步API接口
    • MdsAsyncApi_GetAsyncQueueTotalCount, 返回异步API累计已入队的消息数量
    • MdsAsyncApi_GetAsyncQueueRemainingCount, 返回队列中尚未被处理的剩余数据数量
Since
2019/07/01

◆ F_MDSAPI_ASYNC_ON_CONNECT_T

typedef int32(* F_MDSAPI_ASYNC_ON_CONNECT_T) (MdsAsyncApiChannelT *pAsyncChannel, void *pCallbackParams)

异步API线程连接或重新连接完成后的回调函数的函数原型定义

回调函数说明:

  • 对于TCP通道, 需要通过该回调函数完成行情订阅操作。若函数指针为空, 则会使用通道配置中默认的 订阅参数进行订阅。若函数指针不为空, 但未订阅回报, 90秒以后服务器端会强制断开连接
  • 若回调函数返回小于0的数, 则异步线程将中止运行

线程说明:

  • OnMsg/OnConnect/OnDisconnect 回调函数均运行在异步API线程下
Parameters
pAsyncChannel异步API的连接通道信息
pCallbackParams外部传入的参数
Return values
=0等于0, 成功
>0大于0, 处理失败, 将重建连接并继续尝试执行
<0小于0, 处理失败, 异步线程将中止运行

typedef int32(* F_MDSAPI_ASYNC_ON_DISCONNECT_T) (MdsAsyncApiChannelT *pAsyncChannel, void *pCallbackParams)

异步API线程连接断开后的回调函数的函数原型定义

回调函数说明:

  • 仅用于通知客户端连接已经断开, 无需做特殊处理, 异步线程会自动尝试重建连接

  • 若回调函数返回小于0的数, 则异步线程将中止运行

线程说明:
