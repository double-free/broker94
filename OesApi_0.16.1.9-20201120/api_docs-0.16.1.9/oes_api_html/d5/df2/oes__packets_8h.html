◆ OES_APPL_VER_VALUE

#define OES_APPL_VER_VALUE   1001601091

当前采用的协议版本号数值

  • 版本号数值的格式为 10 位整型数值, 形如: 1AABBCCDDX, 其中:
    • AA 为主版本号
    • BB 为副版本号
    • CC 为发布号
    • DD 为构建号
    • X 0, 表示不带时间戳的正常版本; 1, 表示带时间戳的延迟测量版本
